Для завершения работы по анализу корреляции Кендалла (и Спирмена) для пары случайных величин, было бы хорошо провести такой эксперимент:

Смесь нормального распределения и распределения Стьюдента
f_{mixture)=p*f_{norm}+(1-p)f_{Stud}.

Параметр смеси: p=0,00; 0,05; 0,10;…; 1,00 (с шагом 0,05)

Граф рынка (две вершины, одно ребро).

Коэффициент линейной корреляции 0,5.

Пороги для определения графа рынка: 0,25 (будут ошибки False Negative) и 0,75 (будут ошибки False Positive)

Число наблюдений: 50, 90

Результат: график зависимости вероятности ошибки от параметра смеси.

Четыре способа определять ребро
- Sample Pearson,
- sign similarity
- Kendall correlation
- Spearman correlation ( для корреляции Spermann надо еще повторить эксперимент Lindskog с формулой связи Спирмена и Пирсона как для нормального распределения, см. книгу, для эллиптических эта связь не доказана).
