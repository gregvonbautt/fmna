# Первый эксперимент для лучшего понимания ситуации:
# 
# Имеем сеть с двумя вершинами. Распределение соответствующего случайного вектора (X_1,X_2) как в статье
# Lindskoga t_3 с коэффициентом линейной корреляции 0,5.
# Строим граф отсечения (граф рынка) с переменным порогом  0<\gamma_0<1.
# 
# Генерируем наблюдения (размер выборки можно взять 50 и 90).
# 
# По ним вычисляем оценки корреляции Пирсона тремя способами:
#     - стандартная оценка
#     - через вероятности совпадения знаков
#     - через корреляцию Кендалла
# 
# Сравниваем вычисленную корреляцию с порогом \gamma_0.
# Повторяем эксперимент 3000 раз. Оцениваем (через частоту)  вероятности ошибок
#     - ложно провести ребро (False Positive) и
#     - ложно не провести ребро ( False Negative)
# 
# Строим графики ошибок в осях порог-вероятность ошибки.
# Ожидаемый результат:
# Ошибка со стандартной оценкой Пирсона будет хуже ошибки  через знаки, которая, в свою очередь,
# будет хуже ошибки через Кендалла.
